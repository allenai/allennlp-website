How to Contribute

AllenNLP wants more near state-of-the-art NLP models and is looking for contributions. Not only will your model be listed here and be easily accessible to other users of AllenNLP, but we'll even help build a visualization and host a demo. To contribute a model:

  1. Put your code in a GitHub repo with instructions for running it
  2. Publish a paper or host it on arXiv (ideally)
  3. Host the trained model.tar.gz file somewhere accessible by HTTP
  4. Open an issue with the above links
